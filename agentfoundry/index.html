01 // What AgentFoundry is

An idea should become a product, not a pile of prompts.

AgentFoundry gives Codex a product-building system: start with one useful journey, make the AI safe and durable, then grow from evidence instead of guesswork.

05 // Questions

Before you install

Is AgentFoundry a no-code app builder?

No. It is an open-source skill pack that guides Codex through product and implementation decisions. The result is source code in your repository.

What do I need to start?

Codex and a GitHub repository. Firebase and OpenAI accounts become necessary when the product reaches backend and agent implementation.

Does it include the backend?

Yes. AgentFoundry treats identity, storage, database, server functions, model calls, and external actions as part of the product path.

Do I own the result?

Yes. You keep the repository, source code, product decisions, infrastructure accounts, and brand. AgentFoundry is MIT licensed.

Can I install only one skill?

Yes. Each skill works on its own, while the full sequence covers the path from first proof to continuing product evolution.
