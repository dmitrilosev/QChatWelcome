04 // Questions

Before you install

What is Agent Foundry?

Agent Foundry is an open-source Skill Pack for Codex that turns a product brief into a working AI-native app—with a native interface, backend, AI agents, safety, and testing. You get the source code and product infrastructure in your own accounts and repository, with no platform lock-in.

What do I need to start?

Codex and a GitHub repository. You’ll need Firebase and OpenAI accounts when you are ready to implement the backend and agents.

Does it include the backend?

Yes. AgentFoundry treats authentication, storage, the database, server functions, model calls, and external actions as part of the product journey.

Do I own the result?

Yes. You keep the repository, source code, product decisions, infrastructure accounts, and brand. AgentFoundry is licensed under the MIT License.

Can I install only one skill?

Yes. Each skill can stand on its own, while the full sequence covers the path from the first proof to continuous product evolution.
